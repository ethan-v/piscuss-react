# Piscuss

Piscuss web application build with ReactJS.

![Piscuss](screenshot.png)

**Technical Stack**

- NodeJS 19.x
- Vite 2.x
- React 18.x
- React Router v6
- Bootstrap 5.x
- Fontawesome

## Features

- Lazy Loading

## Quickstart

Install dependencies

```shell
yarn
```

Start development

```shell
yarn dev
```

Build and preview produtction

```shell
yarn build
yarn preview
```
